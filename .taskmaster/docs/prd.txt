# Portfolio Blog Platform - Product Requirements Document

## Overview
A bilingual (EN/VI) portfolio blog platform built with TanStack Start + React 19, deployed on Cloudflare Workers. Features admin approval workflow, Markdown content with syntax highlighting, image uploads to R2, and i18n support.

## Tech Stack
- TanStack Start v1.159.5, React 19, TanStack Router, TanStack Query
- Drizzle ORM + Neon PostgreSQL (serverless)
- Clerk Auth, Cloudflare R2 Storage
- Tailwind CSS 4 + shadcn/ui, Shiki 3 syntax highlighting
- Deployed to Cloudflare Workers (3MB gzip limit)

## User Stories

### US1 - Public Post Reading (P1 MVP) - ~80% COMPLETE
Visitors can discover and read blog posts without authentication.
- Post listing with pagination
- Post detail with Markdown rendering and syntax highlighting
- Language fallback (try requested lang → opposite → 404)
- Translation toggle between EN/VI versions
- Fallback banner when viewing post in non-requested language
REMAINING: PostItem translation badge (T032), localization strings for fallback (T034)

### US2 - Post Creation & Management (P1 MVP) - ~50% COMPLETE
Authenticated authors can create, edit, and delete blog posts.
- Create post with title, slug, content, category, tags, featured image
- Edit existing posts with authorization checks
- Delete posts with confirmation dialog
- Translation workflow (create translation linked via translationGroupId)
- Real-time slug validation
REMAINING: updatePostFn with auth (T037), deletePostFn with auth (T038), edit route (T046-T047), delete dialog (T048-T049), translation UI (T053-T054)

### US2.5 - Admin Post Approval (P1 MVP) - ~70% COMPLETE
Site admin can review and approve/reject submitted posts.
- Approval queue with pending posts
- Approve/reject with feedback
- Submit for approval workflow
- Unpublish functionality
REMAINING: All posts management page (T064), admin badge in header (T071), edit page feedback display (T069-T070)

### US4 - Pagination (P2) - 100% COMPLETE
Post listing with Previous/Next navigation and URL search params.

### US5 - Search & Filter (P3) - NOT STARTED
Full-text search with PostgreSQL, category/tag filtering, search bar, filter components.

### US6 - Responsive Design (P2) - NOT STARTED
Mobile-first responsive layout across all pages and components.

### US7 - Dark Mode (P3) - ~25% COMPLETE

## Database Schema
- users (synced from Clerk)
- posts (bilingual with status enum: draft/pending/published/rejected, translationGroupId)
- categories (1:N with posts)
- tags (M:N with posts via post_tags junction)
- post_tags (composite PK)
